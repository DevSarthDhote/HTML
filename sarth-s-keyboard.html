SHORTCUTS KEYS

Basic Shortcuts: Ctrl + C (or Cmd + C on Mac): Copy
Ctrl + X (or Cmd + X on Mac): Cut
Ctrl + V (or Cmd + V on Mac): Paste
Ctrl + Z (or Cmd + Z on Mac): Undo
Ctrl + Y (or Cmd + Shift + Z on Mac): Redo
Ctrl + A (or Cmd + A on Mac): Select All
Application and Desktop Shortcuts:
Alt + Tab: Switch between open applications
Windows + D: Show Desktop
Windows + L: Lock your computer
Ctrl + Shift + Esc: Open Task Manager
Windows + Print Screen: Take a screenshot of the entire screen

F1 KEYS

The F1 key is primarily used as the Help key in almost all computer programs. When pressed, it typically opens a help screen or directs you to a web page with information about the program. Additionally, F1 can also be used to enter BIOS setup during the computer's boot process, according to Computer Hope. In some programs like Microsoft Excel, Alt + Shift + F1 can be used to create a new worksheet tab.

SPECIAL CHARACTER KEYS

Special character keys on a keyboard are symbols or punctuation marks that go beyond the standard letters and numbers. They're typically found above the number keys, on the right side, and through Alt key combinations. Common examples include !, @, #, $, %, ^, &, *, (, ), -, _, =, +, \, |,, {, }, ;, :, /, ?, ., >, and <. Where to find them: Above Number Keys: Many special characters are located above the numeric keys, requiring you to press the Shift key along with the number key. Right Side of Keyboard: Other special characters are found on the right side of the keyboard, often requiring the Shift key to access them. Alt Key Combinations: You can use Alt key combinations with numeric keypad digits to access a wide variety of special characters and symbols.
